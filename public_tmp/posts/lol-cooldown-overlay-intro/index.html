どんなアプリ？#

LoL Cooldown Overlay は、League of Legends 用の敵スキルクールダウン手動管理オーバーレイです。
いちばん大きな前提は、README にも書いてある通り、ゲームメモリ読み取りや自動操作を一切しないことです。

つまりこのツールは、

  • 自動取得を前提にしない
  • 手動入力で管理する
  • その代わり、画面上で素早く扱える

という方向で作っています。

個人的には、この割り切りがかなり大事でした。
いちばん大きい理由は、Riot Vanguard や規約への抵触を極力避けて、BAN を心配せず使える方向にしたかったからです。
中途半端に自動化へ寄せるより、最初から手動管理に振り切ったオーバーレイとして設計したほうが、方向性がはっきりすると考えました。

何ができるのか#

チャンピオンデータの管理#

このアプリは、まずチャンピオンデータを編集できるところから始まります。

  • 英語名 / 日本語名対応
  • チャンピオンデータの登録、編集、検索
  • お気に入りチャンピオンの上位表示
  • パッチ別データ管理
  • スキルランク別 CD の管理
  • スキルアイコン差し替え

オーバーレイだけ作るのではなく、手元でチャンピオンデータをちゃんと育てられる形にしたかったので、エディタ画面もかなり重要な役割を持たせています。

オーバーレイとしての機能#

実際のオーバーレイ側では、対戦中に必要になる操作へ寄せています。

  • 透過フレームレス表示
  • 常に手前表示
  • スキルボタンのクリックでカウントダウン開始
  • 再クリックでキャンセル
  • レベル帯ごとのスキルランク推定
  • AH (Ability Haste) による CD 短縮計算
  • 設定モードで位置、透過率、サイズ調整
  • ホットキー操作

「一覧管理ツール」ではなく、試合中に見るオーバーレイとして成立することを最優先にしています。

手動管理にした理由#

このアプリのいちばん大きな特徴は、やはり 完全手動 なところです。

手動管理にした理由
  • Riot Vanguard や規約リスクを避けたい
  • BAN を心配しにくい方向へ寄せたい
  • メモリ読み取りや自動化を前提にしない設計にしたい
その代わりに必要だったこと
  • クリックや切り替えが素早くできること
  • レベル帯や AH の調整が楽であること
  • オーバーレイ自体が邪魔にならないこと

手動管理は不便そうに見えますが、規約リスクを避けるための割り切りとして考えると、かなり納得感のある形だと思っています。
だからこそ、UI やホットキー、オーバーレイの軽さを詰めて、手動でも十分使える形に寄せています。

レベル帯と CD 計算#

コード側では、チャンピオンごとの maxOrder とレベル帯情報から、ある程度自然なスキルランクを推定するようにしています。

たとえば通常スキルについては、

  • レベルに応じたスキルポイント数
  • R のランクアップタイミング
  • maxOrder

をもとに、Q/W/E/R の想定ランクを計算します。

さらに、そのうえで

  • Ability Haste
  • CD のオフセット調整
  • 宇宙の英知つき Flash

のような条件も重ねて、オーバーレイ上のカウントダウンへ反映しています。

このあたりは、単なるタイマーではなく、LoL の試合中に見たい CD 情報へ少しでも近づけるために入れている部分です。

画面構成#

エディタ画面#

エディタ画面では、

  • チャンピオン一覧
  • 検索
  • パッチフィルタ
  • お気に入り
  • データ編集
  • 設定
  • オーバーレイ起動

を扱えます。

特に「オーバーレイ起動」セクションは、お気に入りチャンピオンを上に出すようにしているので、よく使う相手をすぐ開けるようにしています。

オーバーレイ画面#

オーバーレイ画面では、

  • チャンピオン名表示
  • サウンド ON/OFF
  • 設定モード切り替え
  • 閉じるボタン
  • レベル帯バー
  • AH 入力
  • CD オフセット
  • カウントダウン音の秒数設定
  • Flash 専用管理
  • スキルボタン群

をまとめています。

設定モードでは透過率とサイズもその場で変えられるので、実際のプレイ画面と重ねたときの邪魔さを減らしやすくしています。

ホットキーと扱いやすさ#

設定タブからキーバインドを変更できるようにしていて、デフォルトでは次の操作が入っています。

  • Ctrl + Shift + S : 設定モード切り替え
  • Ctrl + Shift + X : オーバーレイを閉じる
  • Ctrl + Shift + Right : 次のレベル帯
  • Ctrl + Shift + Left : 前のレベル帯

このあたりは、マウスだけに寄せすぎないために大事にした部分です。
オーバーレイは出せても、操作が重いと結局使わなくなるので、切り替えの速さはかなり意識しています。

技術的な構成#

全体としては、かなりシンプルな Electron アプリです。

UI 側
  • React
  • TypeScript
  • Vite
  • editor / overlay の2画面構成
デスクトップ側
  • Electron
  • main / preload 分離
  • IPC で設定やデータをやり取り
  • ローカル JSON 保存

保存先は %APPDATA%/lol-cooldown-overlay/ 以下で、

  • champions.json
  • settings.json

に分かれています。
データの置き場所が分かりやすいのも、この手のツールでは大事だと思っています。

どういう人に向いているか#

  1. LoL の敵スキル CD を自分で手動管理したい
  2. 自動取得やメモリ読み取りには寄せたくない
  3. 試合画面に重ねる軽いオーバーレイが欲しい
  4. パッチごとにチャンピオンデータを整理して使いたい

逆に、完全自動取得や外部サービスとの連携を前提にするなら、このツールの方向性とは少し違うと思います。

FAQ#

自動で敵スキルを検知するの？
しません。
このツールは 完全手動入力 を前提にしています。
Riot Vanguard や規約への抵触を避け、BAN リスクをできるだけ持ち込まない方向を重視しているためです。
スキルランクは手で全部選ぶ必要がある？
レベル帯に応じたスキルランク推定を入れているので、ある程度は自動で想定できます。
そのうえで AH などを加味して使う形です。
オーバーレイの見た目は調整できる？
できます。
設定モードで位置、透過率、サイズを調整でき、前回の配置も保存・復元されます。

まとめ#

このアプリでやりたかったこと
  • 手動管理に振り切った LoL 用クールダウンオーバーレイを作ること
  • 試合中でも扱いやすい軽い UI にすること
  • レベル帯や AH を踏まえた現実的な CD 表示にすること
  • チャンピオンデータを自分で編集しながら育てられるようにすること

LoL Cooldown Overlay は、派手な自動化を目指すというより、 手動でも気持ちよく使える実用オーバーレイ を目指して作ったアプリです。
同じように、LoL の情報を自分で把握しながら管理したい人にはかなり相性がいいと思っています。
