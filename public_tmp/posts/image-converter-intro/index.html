どんなアプリ？#

Image Converter は、ローカルの画像をすばやく別形式へ変換するために作ったシンプルなブラウザツールです。
ちょっとした画像変換をしたいだけなのに、重いアプリを開いたり、オンラインの変換サイトへアップロードしたりするのが少し面倒だと感じることがありました。

そこで、次のような方向に寄せています。

  • できるだけすぐ使えること
  • サーバーアップロードなしで完結すること
  • PC でもスマホ幅でも扱いやすいこと

個人的には、画像変換は 多機能なエディタ よりも 必要なことだけすぐできる小さな道具 のほうが使いやすい場面が多いと感じています。

何ができるのか#

入力方法がシンプル

画像の読み込みは、普段よく使う 3 パターンに対応しています。

  • 画像ファイルのドラッグ＆ドロップ
  • ファイル選択ボタンからの読み込み
  • クリップボード画像の貼り付け

特に貼り付け対応は、スクリーンショットを取ったあとにそのまま Ctrl + V で持ってこられるので、かなり相性がいいです。

対応形式は絞って分かりやすく

出力形式は、扱いやすさを優先して次の 3 つに絞っています。

  • PNG
  • JPEG
  • WebP

形式を増やしすぎるよりも、日常的によく使うものをすぐ切り替えられること を大事にしたかったので、この構成にしています。

変換前と変換後を見比べられる#

読み込んだ画像と変換後の画像は、どちらもプレビューできるようにしています。
単に保存するだけでなく、

  • どんな画像を読み込んだか
  • 変換後にどうなったか
  • ファイルサイズがどう変わったか

をその場で確認しやすいようにしています。

品質も必要なときだけ触れる#

JPEG と WebP では品質スライダーを調整できます。
一方で PNG は可逆圧縮なので、品質スライダーは意味を持たないため無効にしています。

このあたりは、細かい設定を増やすよりも 必要なときだけ迷わず触れる くらいのバランスを狙っています。

ローカル完結にしている理由#

このアプリは、画像をサーバーへ送って処理する形ではなく、ブラウザの canvas を使ってフロントエンドだけで変換する構成 にしています。

その理由はかなり素直で、

  • ちょっとした変換にサーバーを挟みたくない
  • 個人的な画像をそのままローカルで扱いたい
  • セットアップや運用をできるだけ軽くしたい

という気持ちがあったからです。

画像変換サイトは便利ですが、ローカルにある画像を気軽に処理したいだけなら、アップロード不要でその場で終わる ほうが気楽だと思っています。

実装上の仕様#

JPEG は透過を保持できない#

JPEG は透明度を持てないので、透過 PNG などを JPEG に変換した場合、透明部分は白背景になります。
この挙動は UI 上のヒントにも入れてあって、変換前に分かるようにしています。

PNG は品質スライダーの対象外#

PNG は可逆圧縮なので、JPEG / WebP のような品質調整の考え方とは少し違います。
そのため、このアプリでは PNG を選んだときは品質スライダーを無効にしています。

クリップボード貼り付けはブラウザ依存がある#

貼り付け対応はかなり便利ですが、ブラウザや OS の仕様によっては挙動差が出ることもあります。
そのため、貼り付けが使えない環境でも、ドラッグ＆ドロップやファイル選択で問題なく扱えるようにしています。

技術的な構成#

構成はかなり軽くしています。

フロントエンド
  • React 18
  • Vite 5
  • シンプルな単一画面構成
変換処理
  • FileReader で画像読み込み
  • Image と canvas で描画
  • canvas.toBlob() で形式変換

画像変換は外部ライブラリに大きく寄せるのではなく、ブラウザ標準で扱いやすい範囲を素直に使う 形にしています。
機能は絞っていますが、そのぶん全体の見通しはかなり良い構成です。

セットアップ#

ローカルで試すだけなら、起動もかなり簡単です。

Bash
npm install
npm run dev

Vite の開発サーバーが立ち上がれば、そのままブラウザで動かせます。

どういう人に向いているか#

  1. PNG / JPEG / WebP をすぐ変換したい
  2. オンライン変換サイトへ画像を上げたくない
  3. スクリーンショットを貼り付けてそのまま変換したい
  4. 画像編集ソフトを開くほどではない軽い作業をしたい

逆に、リサイズ、トリミング、色調補正、大量画像の一括処理のような本格的な編集機能が欲しい場合は、今の Image Converter は少し方向性が違います。

FAQ#

入力形式は何でもいい？
基本的には、ブラウザが読み込める画像ファイルであれば扱えます。
README では「ブラウザが読み込める画像ファイル」と「クリップボード上の画像データ」を入力として想定しています。
AVIF などには対応している？
現状は PNG / JPEG / WebP に絞っています。
README にも今後の拡張案として追加形式対応が挙がっています。
サーバーに画像は送られる？
送られません。
画像変換はブラウザ上の canvas で完結する構成です。

まとめ#

このアプリでやりたかったこと
  • 画像変換をもっと軽くすること
  • ローカル完結で扱いやすくすること
  • 貼り付けやドラッグ＆ドロップで流れを短くすること
  • 必要な形式だけを迷わず選べること

Image Converter は、画像編集ソフトの代わりではなく、 「形式を変えたいだけ」の場面を気持ちよく片づけるための小さなツール として作ったアプリです。
ローカルでさっと使える画像変換ツールが欲しい人には、かなり相性がいいと思っています。
