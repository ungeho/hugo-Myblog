## どんなアプリ？

`LoL Cooldown Overlay` は、**League of Legends 用の敵スキルクールダウン手動管理オーバーレイ**です。  
いちばん大きな前提は、README にも書いてある通り、**ゲームメモリ読み取りや自動操作を一切しない**ことです。

つまりこのツールは、

- 自動取得を前提にしない
- 手動入力で管理する
- その代わり、画面上で素早く扱える

という方向で作っています。

個人的には、この割り切りがかなり大事でした。  
いちばん大きい理由は、**Riot Vanguard や規約への抵触を極力避けて、BAN を心配せず使える方向にしたかった**からです。  
中途半端に自動化へ寄せるより、**最初から手動管理に振り切ったオーバーレイ**として設計したほうが、方向性がはっきりすると考えました。

## 何ができるのか

### チャンピオンデータの管理

このアプリは、まずチャンピオンデータを編集できるところから始まります。

- 英語名 / 日本語名対応
- チャンピオンデータの登録、編集、検索
- お気に入りチャンピオンの上位表示
- パッチ別データ管理
- スキルランク別 CD の管理
- スキルアイコン差し替え

オーバーレイだけ作るのではなく、**手元でチャンピオンデータをちゃんと育てられる**形にしたかったので、エディタ画面もかなり重要な役割を持たせています。

### オーバーレイとしての機能

実際のオーバーレイ側では、対戦中に必要になる操作へ寄せています。

- 透過フレームレス表示
- 常に手前表示
- スキルボタンのクリックでカウントダウン開始
- 再クリックでキャンセル
- レベル帯ごとのスキルランク推定
- AH (Ability Haste) による CD 短縮計算
- 設定モードで位置、透過率、サイズ調整
- ホットキー操作

「一覧管理ツール」ではなく、**試合中に見るオーバーレイ**として成立することを最優先にしています。

## 手動管理にした理由

このアプリのいちばん大きな特徴は、やはり **完全手動** なところです。

{{< compare left="手動管理にした理由" right="その代わりに必要だったこと" >}}
- Riot Vanguard や規約リスクを避けたい
- BAN を心配しにくい方向へ寄せたい
- メモリ読み取りや自動化を前提にしない設計にしたい
<!--split-->
- クリックや切り替えが素早くできること
- レベル帯や AH の調整が楽であること
- オーバーレイ自体が邪魔にならないこと
{{< /compare >}}

手動管理は不便そうに見えますが、**規約リスクを避けるための割り切り**として考えると、かなり納得感のある形だと思っています。  
だからこそ、UI やホットキー、オーバーレイの軽さを詰めて、手動でも十分使える形に寄せています。

## レベル帯と CD 計算

コード側では、チャンピオンごとの `maxOrder` とレベル帯情報から、ある程度自然なスキルランクを推定するようにしています。

たとえば通常スキルについては、

- レベルに応じたスキルポイント数
- R のランクアップタイミング
- `maxOrder`

をもとに、Q/W/E/R の想定ランクを計算します。

さらに、そのうえで

- Ability Haste
- CD のオフセット調整
- 宇宙の英知つき Flash

のような条件も重ねて、オーバーレイ上のカウントダウンへ反映しています。

このあたりは、単なるタイマーではなく、**LoL の試合中に見たい CD 情報へ少しでも近づける**ために入れている部分です。

## 画面構成

### エディタ画面

エディタ画面では、

- チャンピオン一覧
- 検索
- パッチフィルタ
- お気に入り
- データ編集
- 設定
- オーバーレイ起動

を扱えます。

特に「オーバーレイ起動」セクションは、お気に入りチャンピオンを上に出すようにしているので、よく使う相手をすぐ開けるようにしています。

### オーバーレイ画面

オーバーレイ画面では、

- チャンピオン名表示
- サウンド ON/OFF
- 設定モード切り替え
- 閉じるボタン
- レベル帯バー
- AH 入力
- CD オフセット
- カウントダウン音の秒数設定
- Flash 専用管理
- スキルボタン群

をまとめています。

設定モードでは透過率とサイズもその場で変えられるので、実際のプレイ画面と重ねたときの邪魔さを減らしやすくしています。

## ホットキーと扱いやすさ

設定タブからキーバインドを変更できるようにしていて、デフォルトでは次の操作が入っています。

- `Ctrl + Shift + S` : 設定モード切り替え
- `Ctrl + Shift + X` : オーバーレイを閉じる
- `Ctrl + Shift + Right` : 次のレベル帯
- `Ctrl + Shift + Left` : 前のレベル帯

このあたりは、**マウスだけに寄せすぎない**ために大事にした部分です。  
オーバーレイは出せても、操作が重いと結局使わなくなるので、切り替えの速さはかなり意識しています。

## 技術的な構成

全体としては、かなりシンプルな Electron アプリです。

{{< compare left="UI 側" right="デスクトップ側" >}}
- React
- TypeScript
- Vite
- editor / overlay の2画面構成
<!--split-->
- Electron
- main / preload 分離
- IPC で設定やデータをやり取り
- ローカル JSON 保存
{{< /compare >}}

保存先は `%APPDATA%/lol-cooldown-overlay/` 以下で、

- `champions.json`
- `settings.json`

に分かれています。  
データの置き場所が分かりやすいのも、この手のツールでは大事だと思っています。
